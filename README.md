项目概述
  这是一个外卖点餐系统，包含 管理端（后台）+ 用户端（前台） 两套业务。

技术栈
  1.前端使用vue;
  
项目定位
  面向餐饮门店的在线点餐/外卖业务系统。
  支持两类角色：
  后台员工/管理员：管理菜品、套餐、分类、订单、员工等。
  普通用户：登录、选餐、加购、下单、地址管理、查看订单。

系统功能
1.  菜单与商品管理  
    菜品分类、单品菜品、套餐，对应后台「分类 / 菜品 / 套餐」页面。
2.  购物车与下单  
    加购、结算、生成订单，对应前台加购、下单、支付成功等流程。
3.  订单管理  
    用户查自己的订单，后台处理/查看订单列表。
4.  收货地址  
    维护配送地址。
5.  账号与权限侧  
    用户短信登录、员工登录与员工维护；后台还有会员列表等页面，本质是围绕「谁在用系统、谁管后台」的支撑能力。
